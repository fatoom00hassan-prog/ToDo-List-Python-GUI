# ToDo List Python GUI

A simple desktop task management application built with Python and Tkinter.

## Project Overview

This project is a task management application that allows users to organize their daily activities through a simple graphical user interface.  
The system helps users keep track of tasks, reduce forgetfulness, and improve productivity.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed or pending
- Display tasks in a clear list
- Store tasks locally so they remain saved after closing the application

## Technologies Used

- **Programming Language:** Python  
- **GUI Framework:** Tkinter  
- **Data Storage:** JSON / CSV  
- **Version Control:** Git & GitHub  
- **Development Tools:** Visual Studio Code / PyCharm


## How to Run

1. Install Python 3
2. Download the project
3. Run the application
